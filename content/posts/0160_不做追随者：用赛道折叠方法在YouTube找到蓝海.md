# 不做追随者：用赛道折叠方法在YouTube找到蓝海

YouTube每个爆火的频道都留下了清晰的路径，只要你会解读就能找到方向。现在的问题是，99%的人都不会读这些痕迹。大部分人看到一个每条视频播放量50万、100万甚至200万的频道，第一反应都是这个频道运气好，算法偏爱它，或者它起步比所有人都早，所以才能拿到流量。

![图片](/images/不做追随者：用赛道折叠方法在YouTube找到蓝海/0.png)

但实际上这个频道能做起来，大概率是因为它选对了内容形式，或者切入了一个很少人做的细分赛道，要么就是同行都没用心做内容，没做出YouTube想要的效果。

你把这些点都分析透了，做内容根本不会愁，这就是我今天要教你的方法。我把这套方法叫做赛道折叠，说白了就是读明白别人留下的成功痕迹，搞清楚已经跑通的模式为什么能成，然后在它的基础上优化。可能是优化内容呈现形式，也可能是切入一个没人做的细分赛道，可调整的地方很多。

# 赛道折叠的实操工具

我把用到的prompt放在这里了，你能直接复制用。

```
`Identify what differentiates this channel from others in the same niche.``Extract recurring visual, narrative and conceptual elements.``Synthesize them into a template sentence that can be replicated in other niches.``Be specific — avoid generic concepts that apply to any educational channel.``Perform this analysis for the channel:  `
```

中文版：

```
找出该频道与同赛道其他频道的差异化特点，提取其反复出现的视觉、叙事与核心理念元素， 将其总结为一句可复制到其他领域的模板化句式， 需具体明确，避免使用所有教育类频道都适用的泛化概念，请对以下频道完成这份分析：
```

我就是靠这套分析方法，筛选出有潜力的细分赛道。用Gemini是因为它是谷歌的产品，能直接调用谷歌和YouTube的全量数据库，分析出来的结果更准确，其他AI的效果不好说。

![图片](/images/不做追随者：用赛道折叠方法在YouTube找到蓝海/1.png)

你只要把频道链接扔进去，就能得到完整的频道分析和应用建议。

![图片](/images/不做追随者：用赛道折叠方法在YouTube找到蓝海/2.png)

为什么说赛道？你看这个示例频道，是用动物形象做虚拟人讲解内容，具体是用大猩猩讲经济、金融相关的内容，这就是这个频道的赛道。

你可能经常听人说什么猴子讲解赛道，其实不对，是用猴子做虚拟人这种形式，切入金融这个赛道。

![图片](/images/不做追随者：用赛道折叠方法在YouTube找到蓝海/3.png)

你看这个频道，总共56条视频，已经拿到了1700万播放，账号做了大概一年时间，收入已经非常不错了。

我让Gemini把这个频道的所有特征都分析了一遍，它用大猩猩和灵长类动物做虚拟人，分别代表不同国家、普通投资者这些角色。把复杂难懂的专业概念，用非常直观、好懂的方式讲出来，比如用香蕉当比喻讲金融知识，搭配大猩猩的形象，普通人很容易就能理解。

这就是这个频道留下的成功痕迹，说明用户喜欢看枯燥复杂的专业内容，被意想不到的虚拟人形象，用日常的普通物品做比喻讲解出来。

这是这个频道用到的成功模式，你去搜一下就会发现，现在已经有很多用大猩猩讲各种内容的账号，数据都不错。

我把这个模式提炼出来，让AI给了我一些通用的赛道方向。当然你如果要真的做，还有更准确的方法，可以进一步验证赛道的潜力。比如我举个例子，量子物理赛道，可以用纸箱做比喻讲解量子纠缠是什么。

![图片](/images/不做追随者：用赛道折叠方法在YouTube找到蓝海/4.png)

封面做起来也很简单，印上小logo，放两个纸箱，写上量子纠缠，再加个箭头就行，纸箱也可以做拟人化处理。

你可能会有疑问，研究这些账号的时候，看到他们的内容制作方式，既然已经摸透了账号的核心逻辑，会做赛道折叠，那我能不能也学他们的剪辑方法，直接把模式复制过来。

完全可以，你甚至可以调整元素，比如把大猩猩换成纸箱，效果说不定更好。现在的AI工具完全能做到，我之前很多视频都讲过分镜脚本的制作，你用AI就能生成对应的素材。

你唯一要做的就是把AI生成的素材，按照分镜脚本拖到剪辑软件里就行，就这么简单。很多人觉得做这个很难，其实根本不是，你把整个制作流程摸透，做起来非常快。

你可能会问，我怎么知道我选的这个主题有没有市场，会不会是在瞎试。和找有潜力的赛道一样，验证主题的方法也很多，你可以做个快速验证。

直接去YouTube搜关键词，比如量子纠缠，你看搜出来的视频，有100万播放的，有接近50万播放的，还有6万播放的，整个量子物理相关的内容播放量都很高。现在用动画加纸箱比喻的幽默形式做这个内容，肯定也能拿到不错的流量。

后面写脚本、做分镜这些都有技巧，但这部分就要靠你自己发挥了，到这一步你就不是在复制别人，而是在做原创内容。

# 避坑和进阶思路

你把我刚才讲的这些方法用到这些热门趋势上，就不是单纯的模仿了。很多人做号只会改个虚拟人形象，内容还是和别人一模一样，你入场晚，只能吃到头部账号剩下的流量。人家11.1万粉丝，内容优先级本来就比你高，平台肯定先推他的内容，人家做的时间久，内容数据更稳定，平台当然会优先给流量。你要是搞不懂这个逻辑，永远只能在YouTube的最底层混。

你可能运气好，刷到个免费教程，做个低门槛或者中等门槛的账号赚到钱了，但等这个赛道饱和了，你又要重新找新方向，重新走一遍流程。就是因为你没花功夫搞懂我最开始讲的内容，不会分析成功模式，不会用到自己的账号里。

质量的账单迟早要付，你不用把精力都花在动画、剪辑这些技术上，但你得搞懂市场，要么帮用户解决问题，要么给喜欢这类内容的用户提供足够的娱乐价值，不然迟早要为质量不足付出代价。
