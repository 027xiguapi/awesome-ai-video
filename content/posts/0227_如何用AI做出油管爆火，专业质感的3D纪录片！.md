# 如何用AI做出油管爆火，专业质感的3D纪录片！

3D纪录片视频在YouTube上很火热，好多新频道的视频播放都能超过百万播放。以前要做出这样的视频，得有多年的动画制作经验才行。现在借助AI，你也可以几分钟内做出看起来很专业的3D纪录片场景。

接下来为你演示如何做一个关于侦探调查悬案的纪录片，以前这种形式的纪录片，要花好几个月才能完成。

第一步：用 ChatGPT 生成精准 3D 提示词（关键！）

打开Chat GPT，输入这个提示：

```
`Create 5 detailed 3D render prompts for a detective cold case documentary. Each prompt should``follow this format: 'A highly stylized 3D render of a featureless human figure with no facial features,``fully smooth and reflective material, in a [scene description with specific props and environment], the``figure is positioned [specific pose/action], wearing [clothing details]. The scene is lit with [lighting``description] casting [shadow details], and the background is [background description]. The figure's``surface is perfectly seamless and ultra-polished, no lines, no seams, no joints, no cracks, no artifacts.``Cinematic angle [camera angle]. Ultra-realistic 3D rendering.' Make each scene relate to detective work``and investigation.`
```

中文版：为侦探悬案纪录片创建五个详细的3D渲染提示。每个提示要按这个格式：高度风格化的3D渲染，无面部特征的人形，表面光滑有反光材质，场景描述包含特定道具和环境。人物摆特定姿势、做特定动作，穿着特定衣服。场景的灯光描述，阴影细节，背景描述。人物表面完美无缝、极度光滑，没有线条、接缝、关节、裂缝和瑕疵。采用电影视角、特定相机角度，超逼真3D渲染。每个场景都要和侦探工作、调查有关。 

Chat GPT就会生成针对这种特定3D风格优化过的详细提示词。看看生成的内容，可不是简单的场景描述，而是完整的提示。

![图片](/images/如何用AI做出油管爆火，专业质感的3D纪录片！/0.png)

可以看到，这格式很适合AI生成，那种独特的无面部特征人形风格很有特色。这样就不用手动去完善每个提示，还能确保做出专业纪录片的效果。

第二步：用 OpenArt 把提示变成 3D 图

接下来，我们要把这些现成的提示变成真正的3D图像。打开OpenArt，点击顶部菜单的“创建图像”。很多人会在这一步犯错，他们用默认模型，其实我们要换成谷歌的Imagine 3，它是专门为高质量3D渲染设计的。点击模型下拉菜单，选择Imagine 3。这个模型在创建有深度和光线效果准确的真实3D图像方面很厉害。

![图片](/images/如何用AI做出油管爆火，专业质感的3D纪录片！/1.png)

然后，把宽高比设为16:9，因为我们要做YouTube内容，在左边的设置面板里能找到这个选项。现在，把ChatGPT生成的第一个提示直接复制到OpenArt里。由于我们已经针对这种特定3D渲染风格优化过了，所以可以直接用ChatGPT生成的内容。把图像数量设为4，这样能有多个选择，然后点击“生成”。看看生成的结果，正是我们想要的。

![图片](/images/如何用AI做出油管爆火，专业质感的3D纪录片！/2.png)

第三步：补不同相机角度，让视频有质感

要让纪录片看起来专业，关键在于同一个场景要有多个不同的相机角度。很多人只做一张图，还奇怪视频为啥没质感。我们回到ChatGPT，让它生成第一个场景的不同相机角度变化提示，输入一下提示词：

![图片](/images/如何用AI做出油管爆火，专业质感的3D纪录片！/3.png)

```
Create 3 different camera angle variations of the detective evidence room office scene using the same featureless figure format: extreme close-up of hands examining files, wide angle showing full investigation room office, and over-the-shoulder shot. Keep the same reflective figure specifications and lighting style.
```

创建侦探调查室场景的三个不同相机角度变化提示：手部检查文件的特写、展示整个调查室的广角、肩扛式拍摄视角。保持同样的反光人物规格和灯光风格。然后在Open Art里生成这些图像，选择flux context max模型，把原始图像设为参考，一个一个粘贴这三个提示词。这样我们就有了同一个场景的三个不同角度，广角、中景和特写，都有人形无面部特征的独特风格，这就是专业纪录片的拍摄方式。

第四步：静态图转动画，秒变电影感

打开 OpenArt 的 “图像转视频” 功能，上传第一张侦探办公室的图，模型选Kling 2.1（其他模型也能试，这个效果最稳）。

重点是 “运动提示” 要贴合场景：比如全景图输 “缓慢推近相机，保持灯光氛围”，时长设 5 秒（够剪又不拖沓）；特写图输 “相机轻微右移，对焦在反光的手和文件上”；广角图输 “静态相机缓慢推近，聚焦前景的案件档案桌”。

![图片](/images/如何用AI做出油管爆火，专业质感的3D纪录片！/4.gif)

这样你就可以做出顶级 3D 场景、独特的无面部人形风格、电影级反光质感、真实 3D 深度，连动画都有专业运镜。更关键的是，每个场景不到 2 分钟生成，所有画面风格还能保持一致。这速度和效果，传统 3D 动画根本比不了。

当然，图片和视频工具现在选择非常多，你不一定用演示中的软件，用你自己习惯使用的平台就可以。
