# 5年35亿播放量 \$500万！揭秘 YouTube爆款频道的重复公式

原来在油管真正赚到Dollar的人也可以很“轻松”。油管博主Riz Rayani分享了他的爆款公式。他在YouTube所有频道总播放量超过35亿，靠的就是这个可以重复和自动化的公式。

他的频道Gamers React 一个关于《我的世界集锦》视频就赚了7万美元。

![图片](/images/5年35亿播放量 $500万！揭秘 YouTube爆款频道的重复公式/0.png)

这个频道发了650多个视频，观看量超过11亿。

![图片](/images/5年35亿播放量 $500万！揭秘 YouTube爆款频道的重复公式/1.png)

![图片](/images/5年35亿播放量 $500万！揭秘 YouTube爆款频道的重复公式/2.png)

还有另外一个频道 Tobey Reacts也是他的，订阅超过3百万，观看量超过16亿。

![图片](/images/5年35亿播放量 $500万！揭秘 YouTube爆款频道的重复公式/3.png)

Gamers React 比Tobey Reacts的收入更高，因为前者全是长视频，而Tobey Reacts有很多短视频，他的频道在大约5年时间里，总共产生了约35亿播放量，在油管获得的收入超过\$500万。

他讲了取得现在成绩的爆款公式。看看他的两个主要频道，Gamers React 和 Tobey Reacts，它们很相似， 这两个频道上反复使用了同一个公式：收集别人爆火的视频二次剪辑。

比如Gamers React ，基本上就是同一个视频反复做，只是每个视频主题不同。像有个视频是一堆《我的世界》的剪辑片段，这些《我的世界》的剪辑片段不是随便找的，找的都是特别精彩的。他找到了一个有效的公式，特定的剪辑风格、特定的《我的世界》片段、解说、配上合适的音乐和各种编辑手法，让视频有趣，然后就反复用这个公式。

![图片](/images/5年35亿播放量 $500万！揭秘 YouTube爆款频道的重复公式/4.png)

Tobey Reacts也是一样，它是个短视频频道，所有短视频基本都是对不同TikTok视频的“解说”，长视频也差不多。

![图片](/images/5年35亿播放量 $500万！揭秘 YouTube爆款频道的重复公式/5.png)

![图片](/images/5年35亿播放量 $500万！揭秘 YouTube爆款频道的重复公式/6.png)

就是这么简单粗暴，但是效果奇佳！

这就是他爆款公式，一个能持续用的策略，并且利用别人的内容。用别人内容时，你得问自己几个问题，能不能持续找到别人的内容。比如Gamers React，他能轻松每周上传四五次视频，因为有足够多的《我的世界》剪辑片段，而且能找到。

很多人问他从哪找《我的世界》剪辑片段，你得找到找这些片段的方法。用于Tobey Reacts 频道的TikTok视频不难找，你只要刷TikTok，收集适合年轻观众的视频就行，之后还能把这活外包出去。

这种方法做法相同，很容易复制。如果每个视频都不同，就没法形成公式，也没法建立流程。Gamers React 他很清楚会做成什么样，因为一直在重复做，所以能建立流程，反复操作。很多人想做不同类型的视频，所以不成功，而他一直做同样的视频。虽然剪辑片段不同，但做每个视频的流程一样，所以能让流程自动化。

这两个频道能有超10亿播放量，另一个原因是有常青内容。常青视频，就是几年后看也有同样效果的视频。比如他最火的视频：<ins>2021《我的世界》精选</ins>，不管是2021年还是2025年看，现在还有很多人看。

![图片](/images/5年35亿播放量 $500万！揭秘 YouTube爆款频道的重复公式/7.png)

而像liberal hivemind这样的频道，是个时政频道，视频时效性强，过了一段时间就没什么人看了。不过，这不是说你一定要做常青内容，选领域时要考虑清楚。选领域很重要，要选能长期做的。比如时政领域能一直做，因为总有新消息。Gamers React 也能一直做，因为总有人想看《我的世界》剪辑片段；Tobey Reacts 面向年轻观众，也有持续的受众。

当然也有靠趋势做起来的频道，像有个专注Diddy（吉普赛人）的频道，有关于Diddy的新闻时就能赚很多钱，但没新闻就没收入，属于“炒作-收割”型频道，这种频道也可以。

但他个人不会做这种频道，他喜欢自动化系统。他以前也做过新闻频道，做久了觉得无聊，虽然能赚钱，但更喜欢能自动化流程化的系统，不用操心明天的新闻。

他组建了团队把这个公式执行得很好，流程基本不用变，因为内容很常青，而且反复做，内容都是差不多的。

还有一个例子，比如Dr Insanity 频道，是犯罪领域的。在这个领域，也能反复做类似的视频，不过标题有点不同，不像Gamers React和Tobey Reacts能用同样的标题。Dr insanity的视频几年后也一样还有人观看，这种频道方向也是不错的。

![图片](/images/5年35亿播放量 $500万！揭秘 YouTube爆款频道的重复公式/8.png)
