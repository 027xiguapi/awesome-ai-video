## 工作流程

```
┌─────────────────┐
│  微信文件       │
│  (wx/...)       │
└────────┬────────┘
         │
         ↓
    ┌─────────────────────────────────────┐
    │  /parse-wx                          │
    │  解析微信导出文件                   │
    │  提取标题、链接、内容等             │
    └────────┬────────────────────────────┘
             │
             ↓
    ┌─────────────────────────────────────┐
    │  JSON数据                           │
    │  (data/parsed/articles.json)        │
    └────────┬────────────────────────────┘
             │
             ├─────────────────────────────────────┐
             │                                     │
             ↓                                     ↓
    ┌──────────────────────┐      ┌──────────────────────┐
    │ /classify-article    │      │ /generate-tags       │
    │ 分类文章             │      │ 生成标签             │
    │ (AI视频/YouTube运维) │      │ (自动提取关键词)     │
    └──────────┬───────────┘      └──────────┬───────────┘
               │                             │
               └──────────────┬──────────────┘
                              │
                              ↓
    ┌─────────────────────────────────────┐
    │ /extract-summary                    │
    │ 提取摘要                            │
    │ (保留核心信息)                      │
    └────────┬────────────────────────────┘
             │
             ↓
    ┌─────────────────────────────────────┐
    │ 处理后的文章数据                    │
    │ (包含分类、标签、摘要)              │
    └────────┬────────────────────────────┘
             │
             ↓
    ┌─────────────────────────────────────┐
    │ /build-knowledge-graph              │
    │ 构建知识图谱                        │
    │ (提取概念、建立关系)                │
    └────────┬────────────────────────────┘
             │
             ↓
    ┌─────────────────────────────────────┐
    │ 知识图谱                            │
    │ (knowledge_graph.json)              │
    │ (节点、边、概念关系)                │
    └────────┬────────────────────────────┘
             │
             ↓
    ┌─────────────────────────────────────┐
    │ Hugo Markdown文件                   │
    │ (content/posts/)                    │
    └────────┬────────────────────────────┘
             │
             ↓
    ┌─────────────────────────────────────┐
    │ 网站                                │
    │ (hugo server)                       │
    └─────────────────────────────────────┘
```

## 数据流

```

   │
   ├─ 微信文件 (.txt)
   ├─ 文章内容 (string)
   ├─ 配置参数 (JSON)
   └─ 用户命令 (CLI)
   │
   ↓
处理层
   │
   ├─ 解析 (parse)

   ├─ 提取 (extract)
   ├─ 生成 (generate)

   ↓
输出数据
   │
   ├─ JSON数据

   ├─ 知识图谱
   ├─ 统计报告

   │
   ↓
应用
   │
   ├─ Hugo网站
   ├─ 知识库
   ├─ 数据分析
   └─ 可视化展示
```

## 快速开始

```bash
# 1. 查看Skills
python skills/skills_manager.py stats

# 2. 处理文件
python skills/skills_manager.py parse-wx wx/articles.txt

# 3. 构建图谱
python skills/skills_manager.py build-graph data/parsed/articles.json

# 4. 查看统计
python skills/skills_manager.py stats --type categories
```

或在Claude对话中：

```
/parse-wx wx/articles.txt
/full-pipeline wx/articles.txt
/stats
```
